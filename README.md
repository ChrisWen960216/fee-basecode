## 1. Curring-Function
+ 函数柯里化: 如果你固定某些参数，你将得到接受余下参数的一个函数（减少函数）
+ 函数柯里化就是---减少参数

## 2. Handlebars
+ `HTML = structure + data`
+ 占位符 + data
+ 类似于 `ES6` 模板字符串

## 3. 列表 (数组)
+ 栈 --- `Stack`
  - 先进后出，后进先出。从末尾压入元素，从末尾弹出元素。

+ 队列 --- `Queue`
  - 先进先出，后进后出。从末尾压入元素，从开头弹出元素。
